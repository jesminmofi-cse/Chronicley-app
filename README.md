# Tech Stack
Frontend: React.js, CSS / Custom styling, Chart.js for visualizations,
Axios for API calls
Backend: Node.js, Express.js, JWT Authentication, RESTful APIs
Database: Mongodb, User-specific data storage, Secure relationships between users and entries

# Security & Privacy
JWT-protected APIs
User-isolated data
No cross-user access
Authentication middleware on all sensitive routes
